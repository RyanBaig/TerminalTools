# EXE

### **_Description_**

Open a local Auto-PY-To-EXE dialog for packaging python projects. I find it super useful for my projects.

### **_Usage_**

1. Open the terminal and type:
```bash
tt --misc exe [ARGS]
# or
tt -m exe [ARGS]
```

### **_Aliases_**

- `tt -misc exe`
- `tt -m exe`
