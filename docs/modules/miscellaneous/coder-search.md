# Coder-Search

### **_Description_**

Searches user queries on GitHub or StackOverflow using Google.

### **_Usage_**

1. Open the terminal and type:

```bash
tt --misc codersearch [ARGS]
# or
tt -m codersearch [ARGS]
```

### **_Aliases_**

- `tt -m codersearch`
- `tt -m cs`
- `tt --misc codersearch`
- `tt --misc cs`

### **_Arguments_**

- **--QUERY (Required)**

**_Example:_**

```bash
tt -m codersearch --QUERY What Are LLMS?
```
